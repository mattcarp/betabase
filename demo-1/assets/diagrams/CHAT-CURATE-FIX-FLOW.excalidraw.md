## Key Flow Points

### 1. Chat Tab (Happy Path)
- User asks question
- System searches vectors (tenant-filtered)
- Returns answer with citations
- Fast, accurate, complete

### 2. Curate Tab (Improvement Loop)
- Collect feedback on all conversations
- Human rates quality, marks sources
- RLHF training improves future responses
- Continuous learning

### 3. Fix Tab (HITL Emergency)
- Complex/ambiguous queries trigger breakpoint
- LangGraph pauses execution
- Human reviews state and provides guidance
- Agent resumes with better context
- System learns from resolution
